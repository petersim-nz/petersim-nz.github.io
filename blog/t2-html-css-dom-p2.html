Web Structure

  • What is grid based design?
    Screen layout is determined by a series of rows and columns which form a grid.
  • What’s all the hype about responsive webpages?
    People use different sized devices (screens) to access the Internet. Webpages should adapt their content to suit the users’ screen sizes.
  • What is semantic structure?
    Semantics is all about meaning. Tags like <h1> or <nav> have a lot more meaning for humans than do tags such as <div> or <span> which give no clues as to their contents.
  • Any other points you found interesting and want to reflect on.
    I hadn’t thought much about adaptive design before. People with tablets or phones are using their thumbs instead of a mouse for navigation and this has implications for the way elements should be laid out.
